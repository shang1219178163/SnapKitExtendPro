v1.0.4 · CocoaPods · SPM

数组约束，一行排布

基于 SnapKit 的 [UIView].snp 扩展： 等间距、固定尺寸、九宫格与反向九宫格，少写重复约束。

iOS 10+ tvOS 10+ macOS 10.12+ SnapKit 5.x MIT
Install 
dependencies: [
    .package(
        url: "https://github.com/shang1219178163/SnapKitExtendPro.git",
        from: "1.0.4"
    )
]
pod 'SnapKitExtendPro', '~> 1.0.4'

API 速览

通过 views.snp 批量约束。输入关键词可即时筛选。

SN
入口

数组约束入口

对 [ConstraintView] 统一 make / update / remake / remove。

views.snp.makeConstraints { make in make.height.equalTo(44) } views.snp.updateConstraints { make in make.height.equalTo(56) }
FX
等间距

固定间距排布

沿水平或垂直方向等间距分布，首尾间距可单独指定。

items.snp.distributeViewsAlong( axisType: .horizontal, fixedSpacing: 10, leadSpacing: 16, tailSpacing: 16 )
SZ
固定尺寸

固定尺寸排布

每个子视图固定宽/高，在父视图内按比例铺开。

items.snp.distributeViewsAlong( axisType: .vertical, fixedItemLength: 48, leadSpacing: 12, tailSpacing: 12 )
SU
九宫格

九宫格 · 固定间距

指定行列间距与每行个数，自动铺满父视图（含边距）。

items.snp.distributeSudokuViews( fixedLineSpacing: 10, fixedInteritemSpacing: 10, warpCount: 3, edgeInset: .init(top: 10, left: 10, bottom: 10, right: 10) )
FW
九宫格

九宫格 · 固定宽高

每个格子固定宽高，按 warpCount 换行排布。

items.snp.distributeSudokuViews( fixedItemWidth: 80, fixedItemHeight: 80, warpCount: 3, edgeInset: .zero )
RV
反向

反向九宫格

从右向左填充的九宫格，适合 RTL 或特殊视觉顺序。

items.snp.distributeSudokuViewsReverse( fixedLineSpacing: 10, fixedInteritemSpacing: 10, warpCount: 3, edgeInset: .zero )
HZ
水平流

水平流式排布

水平串联、垂直居中；支持正向 / 反向，子视图宽度自适应内容。

labels.snp.distributeViewsHorizontal( fixedSpacing: 8, leadSpacing: 16, tailSpacing: 16, isReverse: false )
